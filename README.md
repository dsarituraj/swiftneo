Eco-Friendly Car E-Commerce Website

A modern and interactive e-commerce website focused on eco-friendly vehicles, designed to deliver a smooth and engaging shopping experience with advanced UI/UX and animations.

🔗 Live Demo: https://error-404-reimagine-round2.vercel.app/

Features

Responsive design for all devices (mobile-first approach)
Interactive product browsing with filtering and search
Dynamic shopping cart with real-time updates
Smooth animations using GSAP
Parallax scrolling with Locomotive Scroll
Modern UI with custom shader-based visual effects
Optimized performance with lazy loading and efficient rendering

What Makes This Project Special
Built a real-world production-like UI, not a basic college project
Focused on user experience, animations, and visual storytelling
Implemented modern frontend practices used in top companies
Designed to match industry-level websites like Tesla

Tech Stack
Frontend: HTML, CSS, JavaScript
Animations: GSAP (GreenSock)
Scrolling: Locomotive Scroll
Graphics: WebGL, Shaders (GLSL)
Version Control: Git & GitHub

To build a visually appealing and high-performance e-commerce platform that enhances user experience through smooth animations, responsive design, and intuitive navigation.

Backend integration (Node.js, MongoDB)
Authentication system
Payment gateway integration
AI-based recommendations
